# Primary color

This is the main color used the most through out the web app. Sentinel digital's primary color is an orange color with the hex code of #CB7B00 . This color is used in places like the app header and footer, circular icons, headers, overlines etc.

# Secondary color

The secondary color is the second most widely used color throughout the web app. Sentinel's secondary color is purple #372189 because purple on the color wheel is complimentary to orange and therefore balances out the primary color

# Shades and tints

Several gradations of shades and tints are used to achieve a sense of variety from the primary and secondary colors. These shades and tints are used both in UI elements as well as for text elements.

# Tints

Tints are achieved by adding white to a color. The lightest tint is has 95% white applied to it and the darkest tint has only 20% white added to it.

# Special colors

For certain use cases, there are a couple of colors that fall outside of the general color scheme. These colors different because of how they are traditionally perceived. The color for web links are traditionally a vibrant blue and is easily recognizable as such by users. The same goes for red as an alert color. Both colors require little to no explanation when used in a web context and therefore makes them an obvious choice for alerts and links respectively.
